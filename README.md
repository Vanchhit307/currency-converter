# Currency Converter

Java Swing/AWT application for converting between 12 major world currencies with fixed exchange rates.

## Features

- Dropdown menus for currency selection
- Fixed exchange rates (relative to USD)
- Swap currencies functionality
- Real-time exchange rate display
- Input validation (numbers only)
- Keyboard shortcut: Press Enter to convert
- Clear button to reset fields
- Currency symbols display ($, €, £, ₹, etc.)

## How to Use

1. Select source currency from "From Currency" dropdown
2. Select target currency from "To Currency" dropdown
3. Enter the amount to convert
4. Click "Convert" button or press Enter key
5. View the converted amount
6. Use "Swap Currencies" to reverse the conversion
7. Click "Clear" to reset all fields

- 5000 INR = 59.88 USD
